AI Quest: Think Like a Smart AI

AI Quest is a brutalist, story-driven educational web application designed to teach the fundamentals of Mean-Ends Analysis (MEA).

It delivers an immersive, multi-level experience where users solve challenges by applying structured problem-solving techniques. Each level reinforces core MEA concepts such as identifying differences, selecting operators, and achieving subgoals.

A sarcastic AI guide adds a meme-driven, troll-style interaction layer, making learning both engaging and memorable.
